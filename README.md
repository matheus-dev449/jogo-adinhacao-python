# 🎯 Jogo de Adivinhação

Este é um simples jogo de adivinhação desenvolvido em Python, onde o jogador deve tentar acertar um número secreto gerado pelo programa.

## 📌 Sobre o projeto

O objetivo do jogo é desafiar o usuário a adivinhar um número aleatório dentro de um intervalo definido. A cada tentativa, o programa informa se o número digitado é maior ou menor que o número secreto.

## ⚙️ Tecnologias utilizadas

- Python 3
- Biblioteca `random`

## 🎮 Como funciona

- O programa gera um número aleatório.
- O usuário tenta adivinhar o número.
- O sistema dá dicas se o palpite é maior ou menor.
- O jogo continua até o usuário acertar.
